# SlipStream GUI Features

## Core Features
*   **VPN Connection**: Connect to SlipStream servers using a user-friendly toggle.
*   **System Proxy Integration**: Automatically sets system proxy settings (optional).
*   **Cross-Platform**: Windows, macOS, and Linux support.

## Configuration Management
*   **Multiple Profiles**: Save multiple server configurations with custom remarks and country flags.
*   **Config Selection**: Easily switch between saved configurations from the Home page.
*   **SOCKS5 Authentication**: Support for username/password authentication for SOCKS5 proxies.

## DNS Tools
*   **DNS Tester**: Built-in tool to test latency of multiple DNS servers.
*   **Performance Metrics**: View Ping and DNS resolution times.
*   **One-Click Apply**: "Use" button to immediately apply a fast DNS server.
*   **Authoritative Mode**: Enforce strict DNS resolution for bypassing strict firewalls.

## User Interface
*   **Modern Design**: Clean, semi-transparent glassmorphism aesthetic.
*   **Real-time Stats**: (Simulated) Traffic download/upload speed indicators.
*   **Keyboard Accessibility**: Fully accessible form controls.
*   **Responsive**: Adaptive layout for different window sizes.

## Advanced
*   **Verbose Logging**: detailed logs for debugging.
*   **Local Proxies**: Exposes HTTP (8080) and SOCKS5 (10809) ports for local device usage.
